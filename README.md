🙋‍♀️ 우리 서비스는요...

1. 냉장고의 **식재료를 관리**해줘요!

3. 냉장고를 공유하는 그룹원에게 먹고 싶은 **레시피를 요청**할 수 있어요!

## 🐳 Docker Compose

`application.properties`, `docker compose script` 환경변수(`.env`) 설정 후 다음 명령어로 컨테이너에서 프로젝트를 build해볼 수 있어요!

```shell
docker-compose -f docker-compose-ci.yml up -d
```
